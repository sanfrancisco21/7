Projects and Publications

Artsy Robot

The Artsy Robot was a project that I did for my Mechatronics class a while back. It started off as a simple autonomous robot that was able to back up and turn to avoid collisions (note the distance sensor on the front). Later on, I wrote a program that allowed my robot to follow a premade path. By attaching a marker to the back, I was able to program the robot to draw simple (i.e. continuous) shapes and patterns.
